# [B] 자금 계획 규칙

당첨 시 자금 계획 계산에 필요한 규칙. 자금 계산기(`lib/core/finance/`)가 이 문서를 반영한다. (배경·용어는 [overview.md](./overview.md), 자격은 [eligibility.md](./eligibility.md))

> ⚠️ 자금/대출 추정은 참고용이며 최종은 금융기관 확인이 필요하다 — 결과엔 면책 고지 동반.

## 1. 납부 스케줄 규칙

> 🚧 미작성 — 계약금/중도금(회차)/잔금의 구성과 계산. 비율·금액·납부일은 공고에서 파싱(공고 변수), 일반적 관례(계약금 10~20%, 중도금 60% n회 분납 등)는 여기에 기준으로 정리.

## 2. 대출 가능액 규칙

> 🚧 미작성 — 대출 가능액 = **min(LTV 한도, DSR 한도, 중도금대출 규정)**. 자기자본 필요액 = 분양가 − 대출.

### 2.1 LTV 비율표 — 규제지역·분상제·주택가격·무주택 여부별
### 2.2 DSR 한도 — 연 원리금/연소득 상한 (은행권)
### 2.3 중도금대출 규정 — 보증 한도·세대당 건수 등
### 2.4 가정값 — 대출 금리·만기 (기본값 제공, 사용자 조정 가능)

## 3. 계산식

> 🚧 미작성 — 납부 스케줄, DSR 원리금(금리·만기 기반), LTV/DSR/중도금 결합으로 대출 가능액·자기자본 산출하는 식.
