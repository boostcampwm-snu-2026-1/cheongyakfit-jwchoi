# 도메인 배경 + 용어집

청약 도메인의 배경 지식과, "공고 변수 vs 법령 고정"의 구분, 용어집. (전체 안내는 [README](./README.md))

## 1. 민영 청약 공급 구조

민영주택 청약은 **일반공급**과 **특별공급(특공)** 으로 나뉜다.

- **일반공급**: 가점제(무주택기간·부양가족수·통장가입기간 합산) + 추첨제. **소득 조건 없음.**
- **특공 5종**: 신혼부부·생애최초·다자녀·노부모부양·신생아. 유형마다 자격 조건 상이.
- 기관추천 특공(국가유공자·장애인 등)은 자동판정 불가 → 안내만.

## 2. 공고 변수 vs 법령 고정 기준

공고마다 달라지는 것과 법령(주택공급에 관한 규칙)으로 고정된 것을 명확히 구분한다.  
이 구분이 **파싱(LLM) ↔ CORE 엔진(순수 함수) 분리**의 근거다.

### 공고마다 달라지는 것 → PDF 파싱 대상

| 요인 | 설명 |
|------|------|
| 분양가 | 평형(전용면적)별 상이 — 공고 요약 표시용 |
| 공급 세대수 | 특공/일반, 가점/추첨, 평형별 세대수 |
| 평형 구성 | 전용 59㎡·84㎡·101㎡ 등 단지마다 다름 |
| 청약 일정 | 모집공고일·접수기간·당첨자 발표일·계약기간 |
| 입주예정일 | |
| **규제지역 여부** | 투기과열지구 / 청약과열지역 / 일반 — 이하 파생 조건의 결정 변수 |
| **분양가상한제 적용 여부** | 전매제한 기간·실거주 의무의 결정 변수 |
| 청약 가능 지역 범위 | 1순위 해당 시·도 우선, 인근 포함 여부 등 |
| 전매제한 기간 | 공고에 명시되지만 위 두 변수로 계산도 가능 |

### 입지 변수에서 파생되는 조건 (규칙은 법령 고정, 해당 여부만 공고마다 다름)

| 결정 변수 | 파생 조건 |
|-----------|----------|
| 규제지역 여부 | 가점제·추첨제 비율, 1순위 통장 가입기간(2년/1년/6개월), 재당첨 제한 기간, 신혼·생애최초 소득기준 % |
| 분양가상한제 여부 | 전매제한 기간, 실거주 의무 기간 |
| 전용면적 구간 | 85㎡ 이하/초과별 가점·추첨 비율 |

### 법령으로 고정된 것 → DB rule 테이블 (값의 런타임 SSoT, 구조·근거는 이 문서)

PDF에서 뽑지 않고 룰로 관리한다. 단 코드 하드코딩이 아니라 **DB rule 테이블**에 두고 엔진이 로드해 주입받는다(자격 요건·법령이 자주 바뀌므로 — [decisions.md](../decisions.md) D16). 이 문서는 룰의 **구조·근거·출처**의 SSoT.

| 항목 | 비고 | 정의 위치 |
|------|------|----------|
| 특공별 소득 기준 % | 도시근로자 월평균 소득의 몇 % 이하 — % 자체는 불변, 절대 금액은 매년 갱신 | [eligibility.md](./eligibility.md) |
| 가점 산정 방식·점수표 | 무주택기간(≤32) + 부양가족수(≤35) + 통장가입기간(≤17) = 84점 | [eligibility.md](./eligibility.md) |
| 예치금 기준표 | 지역·전용면적 조합별 고정 금액 | [eligibility.md](./eligibility.md) |
| 특공 기본 자격 요건 | 무주택 여부, 혼인기간, 자녀수 등 | [eligibility.md](./eligibility.md) |

> **소득 기준 절대 금액(원)은 매년 1회 갱신** — 통계청 도시근로자 가구당 월평균 소득 발표 시점.  
> rule 테이블 시드/마이그레이션 갱신 시 역참조 주석에 기준연도·출처 명시 필수.
