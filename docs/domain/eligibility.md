# 자격 판정 규칙

청약유형별 판정 룰 + 자격 기준표(소득/가점/예치금) + 서류 매핑. 매칭 엔진(`lib/core/matching/`)이 이 문서를 반영한다. (배경·용어는 [overview.md](./overview.md))

> **이 문서는 룰의 구조·근거·출처의 SSoT.** 기준표 **값**의 런타임 SSoT는 **DB rule 테이블**이다(엔진이 로드해 주입 — [decisions.md](../decisions.md) D16). 표의 값은 시드/마이그레이션으로 옮기고 역참조 주석을 단다.

## 0. 전제·기준연도·출처

- **대상**: 민영주택(민간분양) 자격 판정만. 국민주택·공공분양/임대는 범위 밖. 기관추천 특공(국가유공자·장애인 등)은 자동판정 불가 → "해당 가능성 있음, 별도 확인" 안내만.
- **법령 근거(최상위)**: [주택공급에 관한 규칙 (국가법령정보센터)](https://www.law.go.kr/LSW//lsInfoP.do?lsId=008243). 아래 모든 룰은 이 규칙과 그 별표를 따른다.
- **기준연도**: 소득표는 **2025년 적용분(2024년 귀속 통계)**. 가점·예치금·배점은 현행 규칙 기준. 소득 절대금액은 매년 1회 갱신(통계청 도시근로자 월평균소득 발표 시).
- **최근 개정(반영됨)**:
  - **2025-10-16 「10·15 주택시장 안정화 대책」 시행** — 규제지역 대폭 확대(서울 전역·경기 12곳), 규제지역 1순위를 **무주택 세대주**로 제한, 가점 비중 상향. ([국토교통부 보도자료](https://www.molit.go.kr/USR/NEWS/m_71/dtl.jsp?id=95091308), [한국경제](https://www.hankyung.com/article/2025110926271))
  - **2026-06-15 민영주택 「신생아 특별공급」 신설 시행** — 특별공급 물량의 10% 배정. ([서울경제](https://www.sedaily.com/article/20055422), [한국경제](https://www.hankyung.com/article/202606144850i))
  - **2024-03-25 다자녀 특공 기준 완화** — 미성년 자녀 3명 이상 → **2명 이상**. ([찾기쉬운 생활법령정보](https://easylaw.go.kr/CSP/CnpClsMain.laf?popMenu=ov&csmSeq=1126&ccfNo=3&cciNo=1&cnpClsNo=1))
- **⚠️ 표시**: 별표의 정확한 숫자(공급 배분 %, 다자녀 배점 등)는 DB rule 시드 단계에서 해당 회차 입주자모집공고·규칙 별표로 **재확인**한다. 이 문서는 구조·출처의 기준점.

## 1. 청약유형별 판정 룰

각 유형: **자격 조건(법령 근거)** / 조건이 읽는 **입력값**(프로필·공고) / **필요 서류**.

### 공통 전제 (민영 6종 모두)

| 조건 | 내용 | 읽는 입력값 |
|------|------|------------|
| 청약통장 | 주택청약종합저축 가입 + 신청면적 **예치금**(§2.3) 충족 | 프로필: 통장 보유·가입일·예치금 / 공고: 신청면적·지역 |
| 무주택 | 일반공급 가점제·전 특공은 **무주택세대구성원**(노부모부양은 무주택**세대주**) | 프로필: 세대원 주택소유 / 공고: — |
| 재당첨 제한 | 과거 5년(규제지역) 이내 당첨 세대 구성원이 아닐 것 | 프로필: 과거 당첨 이력 / 공고: 규제지역 여부 |
| 특공 1회 | 특별공급은 세대당 평생 1회 | 프로필: 특공 당첨 이력 |

> 규제지역 여부·신청면적·공급유형별 세대수는 **공고 파싱**으로 확보(=공고 변수, [overview.md](./overview.md) §2). 나머지 자격은 프로필·룰로 판정.

### 1.1 일반공급 (가점제 / 추첨제)

**1순위 요건** — 출처: [정부24 주택청약 가점제도](https://www.gov.kr/portal/service/serviceInfo/PTR000051707), [주택공급규칙](https://www.law.go.kr/LSW//lsInfoP.do?lsId=008243)

|------|-----------|----------------------------|------------|
| 통장 가입기간 | 수도권 1년 / 비수도권 6개월 (위축지역 1개월) | **2년 경과** | 프로필: 통장 가입일 / 공고: 규제·지역 |
| 예치금 | 신청면적별 기준액 이상(§2.3) | 동일 | 프로필: 예치금 / 공고: 면적·지역 |
| 세대 요건 | 제한 없음(세대원도 1순위 가능) | **무주택 세대주만** (10·15 대책) | 프로필: 세대주 여부·세대 무주택 |
| 재당첨 | — | 과거 5년 내 당첨 세대 아님 | 프로필: 당첨 이력 |

**당첨자 선정 — 가점제/추첨제 비율** (전용면적 × 규제지역). 출처: [주택공급규칙](https://www.law.go.kr/LSW//lsInfoP.do?lsId=008243), [당첨자 선정 기준 안내](https://www.hobansummit.co.kr/sell/information_01/web)

| 전용면적 | 투기과열지구 | 조정대상지역 | 비규제지역 |
|---------|------------|------------|-----------|
| 60㎡ 이하 | 가점 40 / 추첨 60 | 가점 40 / 추첨 60 | 가점 ≤40(지자체 0~40 자율) / 추첨 나머지 |
| 60㎡ 초과 ~ 85㎡ 이하 | 가점 70 / 추첨 30 | 가점 70 / 추첨 30 | 가점 ≤40 / 추첨 나머지 |
| 85㎡ 초과 | 가점 80 / 추첨 20 | 가점 50 / 추첨 50 | 가점 0 / 추첨 100 |

(단위 %. 가점제는 §2.2 가점으로 줄세움, 추첨제는 무작위.)

- **읽는 입력값**: 프로필 — 무주택기간·부양가족수·통장가입기간(가점 계산), 세대주·세대 무주택(규제지역). 공고 — 규제지역 여부·전용면적·가점/추첨 세대수.
- **판정 출력**: 1순위 자격 충족 여부 + (가점제 해당 시) 산정 가점. 가점은 점수일 뿐 당첨 보장 아님 → 결과는 "가능(자격 충족)"으로 판정하고 가점은 참고 표기.

- **필요 서류**: §3 — 혼인관계증명서, 등본, 가족관계증명서, (해당 시)임신진단서/입양관계증명서, 소득·자산 증빙.

## 3. 서류 매핑 (조건 ↔ 증빙)

판정 근거 조건마다 그것을 증명하는 서류. 결과 화면의 `requiredDocuments[]`는 **실제로 적용된 판정 조건**에서만 생성(조건↔서류 1:1). 출처: [청약홈 — 특별공급 안내](https://www.applyhome.co.kr/ar/ara/selectSubscrptIntroSpetialView.do), [LH청약플러스 — 제출서류](https://apply.lh.or.kr/lhapply/cm/cntnts/cntntsView.do?mi=1226&cntntsId=1113).

|----------|----------|----------|
| 무주택 여부·기간 | 주민등록표 등본·초본, (필요 시) 지방세 세목별 과세증명·건축물대장 | 전 유형 |
