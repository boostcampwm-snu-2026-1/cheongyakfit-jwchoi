# [A] 자격 판정 규칙

청약유형별 판정 룰 + 자격 기준표(소득/가점/예치금) + 서류 매핑. 매칭 엔진(`lib/core/matching/`)이 이 문서를 반영한다. (배경·용어는 [overview.md](./overview.md), 자금은 [finance.md](./finance.md))

## 1. 청약유형별 판정 룰

> 🚧 미작성 — 유형마다: 자격 조건(법령 근거) / 조건이 읽는 입력값(프로필·공고) / 필요 서류(조건↔증빙 1:1).

### 1.1 일반공급 (가점제 / 추첨제)
### 1.2 신혼부부 특별공급  ← **여기부터 시작**
### 1.3 생애최초 특별공급
### 1.4 다자녀가구 특별공급
### 1.5 노부모부양 특별공급
### 1.6 신생아 특별공급

> 기관추천 특공(국가유공자·장애인 등)은 자동판정 불가 → "해당 가능성 있음, 별도 확인 필요"로만 안내.

## 2. 자격 기준표 (Reference Tables)

> 🚧 미작성 — 각 표에 출처·기준연도 명시. 코드 `lib/core/rules/`가 이 표를 반영.

### 2.1 소득기준표 (신혼·생애최초) — 가구원수별 / 맞벌이 여부별
### 2.2 가점표 — 무주택기간(≤32) / 부양가족수(≤35) / 통장가입기간(≤17) = 84점
### 2.3 예치금표 — 지역·면적별

## 3. 서류 매핑 (조건 ↔ 증빙)

> 🚧 미작성 — 판정 근거 조건마다 그것을 증명하는 서류.
